# spring-boot-project-samples

#### 介绍
常用的SpringBoot工程案例，比如集成短信服务、OSS服务、Redis缓存、Quartz调度框架、SpringSecurity安全框架等，
以及JSR303校验的补充、加解密功能、动态数据源、excel操作等。
